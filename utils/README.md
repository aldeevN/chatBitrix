# Telegram-like Bitrix24 Chat with Authentication and Real-time WebSocket

## Описание

Это приложение представляет собой клиент для Bitrix24 с интерфейсом, похожим на Telegram. Оно включает:

- Аутентификацию через Selenium с захватом cookies и конфигурации
- Реальный чат через WebSocket (Bitrix Pull)
- Telegram-подобный интерфейс с темной и светлой темами
- Отправку и получение сообщений в реальном времени
- Загрузку и отправку файлов

## Особенности

- **Аутентификация**: Автоматический захват данных аутентификации из браузера
- **WebSocket**: Реальное время через Bitrix Pull протокол
- **Telegram UI**: Современный интерфейс с пузырьками сообщений
- **Темы**: Поддержка светлой и темной тем
- **Файлы**: Поддержка вложений с иконками по типу файла
